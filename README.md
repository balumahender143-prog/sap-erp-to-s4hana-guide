# SAP ERP → S/4HANA Universe 🚀

An interactive, single-page dashboard that explains **SAP S/4HANA fundamentals** — covering 30+ modules across Finance, Logistics, Manufacturing, HR, Procurement, and next-generation AI. Built for learners, consultants, and anyone moving from classic SAP ECC to S/4HANA.

Each module card includes a real-world example, the **standard process flow logic** (with actual SAP transaction codes), and a direct link to **official SAP Learning** content.

## 🌐 Live Demo

**👉 [View the live dashboard](https://balumahender143-prog.github.io/sap-erp-to-s4hana-guide/)**

> Replace `Balu Mahender` with your GitHub username once GitHub Pages is enabled.

## ✨ Features

- **9 tabbed sections** organizing 38 module cards by business area
- **Dark & light mode** toggle (top-right)
- **Standard flow logic** on every module — step-by-step process chains with real transaction codes (VA01, ME21N, MIGO, IW31, and more)
- **Clickable module cards** that open the best-matching official SAP Learning page
- **Real-world examples** and key tables / master-data notes per module
- **Joule AI** interactive chat demo
- **Animated** progress bars, metrics, and an SVG module universe
- Fully **responsive** — works on desktop and mobile

## 🚀 Run Locally

Just open the file in any browser:

```
index.html
```

No server or installation required.

## ☁️ Deploy on GitHub Pages

1. Make sure the HTML file is named **`index.html`**
2. Go to **Settings → Pages**
3. Under **Build and deployment**, set **Source: Deploy from a branch**
4. Select the `main` branch and `/ (root)` folder, then **Save**
5. Your site goes live at `https://YOUR-USERNAME.github.io/sap-erp-to-s4hana-guide/`

## 📖 Learning Links

Every module card links to official **[SAP Learning](https://learning.sap.com/)** topic pages and learning journeys. Note that SAP occasionally restructures its site, and some learning journeys may require a free SAP Learning sign-in.
